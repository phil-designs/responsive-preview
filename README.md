## About The Project

[![Product Name Screen Shot][product-screenshot]](https://github.com/phil-designs/responsive-preview)

Responsive Preview adds a link to the WordPress admin toolbar that opens your site inside a full-screen preview interface. Switch between device viewport sizes and use the Responsive Walk feature to animate a smooth sweep from 1440px down to 320px.

**Features:**

* Device Viewports — desktop, laptops, iPad, iPhones, Pixel, and more; portrait/landscape toggle on rotatable devices
* Custom Devices — define your own presets from the settings page
* Responsive Walk — animated sweep from 1440px to 320px with scrubbable progress bar
* Settings page — enable or disable any viewport from the dropdown

<!-- GETTING STARTED -->
## Getting Started

 

### Installation

1. Upload the 'responsive-preview' folder to '/wp-content/plugins/'
2. Activate the plugin through the Plugins menu in WordPress
3. Click Responsive Preview in the admin toolbar to open the preview
4. Go to the Settings>Responsive Preview page to add viewports


<p align="right">(<a href="#readme-top">back to top</a>)</p>

<!-- USAGE EXAMPLES -->
## Usage

* Click Responsive Preview in the admin toolbar — a new window opens.
* Use SELECT VIEWPORT to switch device sizes.
* Click the orange play button at the bottom to start a Responsive Walk.
